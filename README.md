# Personal Finance Tracker

A web-based Personal Finance Tracker built using Flask, SQLite, Bootstrap, and Matplotlib.

## Features

- Add Income and Expenses
- Dashboard with Total Income, Expense, and Balance
- Expense Analysis Pie Chart
- Delete Transactions
- User Login & Registration
- Responsive Bootstrap UI

## Technologies Used

- Python
- Flask
- SQLite
- Bootstrap 5
- Matplotlib
- HTML/CSS

## Run Project

```bash
pip install flask matplotlib
python app.py
